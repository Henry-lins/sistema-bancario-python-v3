# Sistema bancario em Python-v3
Este projeto é um simples sistema bancário desenvolvido em Python, com o objetivo de praticar lógica de programação e POO.
# Funcionalidades
- Depósito
- Saque
- Extrato
- Criar contas
- Listar contas
- Criar clientes
- Sair do sistema
# Conceitos
-POO
-Classe e herança
-Dados
-Estruturas condicionias
# Obs:
Este projeto foi desenvolvido para estudos e está em constante evolução.
